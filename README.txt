Water Grid — app simples para controle de regas
================================================

Como usar (rápido):
1) Descompacte a pasta.
2) Abra o arquivo `index.html` no celular ou computador (não precisa de internet).
3) Toque curto em um quadradinho para marcar que regou aquela área (inicia o cooldown).
4) Toque e segure em um quadradinho para editar nome e cooldown daquela área.
5) Clique em ⚙️ para abrir mais configurações (linhas/colunas, imagem do terreno, etc.).
6) Exporte/importe os dados quando quiser fazer backup.

Dica de imagem:
- O grid padrão é 10×5 (50 áreas) com quadrados perfeitos se a imagem for 2:1 (duas vezes mais larga que alta).
- Se quiser outro arranjo, mude Linhas/Colunas nas Configurações (os dados existentes são preservados quando possível).

Instalar como app (PWA):
- Hospede estes arquivos em qualquer serviço estático (GitHub Pages, Netlify, Vercel).
- No navegador do celular, abra a URL e use “Adicionar à tela inicial”. Passa a funcionar offline.

Tudo é salvo localmente no navegador (LocalStorage).

— Feito para você cuidar da chácara com praticidade.
